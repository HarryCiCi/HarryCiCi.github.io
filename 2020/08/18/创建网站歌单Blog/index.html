1.hexo-tag-aplayer 是Aplayer在hexo上的插件，这里的配置参考的是官方文档，第一步安装 hexo-tag-aplayer：

npm install --save hexo-tag-aplayer

2.最新版的 hexo-tag-aplayer 已经支持了MetingJS的使用，可以直接解析网络平台的歌曲（简直是神器），首先要在站点配置文件中开启meting模式，添加以下代码在配置文件的最后：

aplayer:
  meting: true

3.复制歌单的链接，然后复制歌单的id，例如 https://music.163.com/playlist?id=5187485539 ，这个歌单的id就是5187485539，公司名可以是tencent、netease或是其他公司，下面给出一个例子，打开 E:\Blog\source\music\index.md文件，输入：

{% meting "5187485539" "netease" "playlist" "theme:#2234" "mode:circulation" "mutex:true" "listmaxheight:340px" "preload:auto" %}

添加导航项的方法

1.导航栏音乐菜单可以新建：对应站点配置文件和主题配置文件_config.yml


2.主题文件代码设置：

<% if(page.type === "music") { %>
      <!-- 音乐页 url形式：https://yoursite/about/ -->
        <div class="container about-index">
          <div class="markdown-body">
            <%- page.content %>
          </div>
        </div>
      <% } %>
